# Instructions for French Verb Conjugation Files

## GENERAL FORMAT RULES
1. Each verb must begin with a level-2 heading (##) containing:
   - the infinitive form in French,
   - the IPA transcription in square brackets,
   - an equals sign followed by the English meaning in lowercase.

   Example format:  
   `## Sentir [sɑ̃tiʁ] = to feel`

2. After the heading, include a Markdown table with exactly two columns:
   | Conjugation | Pronunciation |
   These headers must appear exactly as written.

3. The conjugation column must list all six standard persons in this order:
   - Je …
   - Tu …
   - Il/elle …
   - Nous …
   - Vous …
   - Ils/elles …

4. The pronunciation column must follow IPA

5. After each verb table, insert a single horizontal rule:


6. No blank lines may be added except:
   - one blank line before the next verb heading,
   - the blank line required before the horizontal rule.

## CONTENT RULES
- Reflexive verbs must always include the reflexive pronoun in the conjugation column (e.g., Je m’améliore).
- The pronunciation column for reflexive verbs must reflect correct reflexive pronunciation.
- Liaisons must be shown with underscores (e.g., nu nu_zameljɔʁõ).
- Use “Il/elle” and “Ils/elles” exactly in this format.
- Do not add English translations inside the table.
- Do not insert explanations or commentary between sections.

- Keep spacing consistent with the examples.
- Each verb must be presented in its own block following this exact format.
